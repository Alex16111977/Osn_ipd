## 1. КРИТИЧНІ ПОМИЛКИ: Неіснуючі коди

### 1.1 Код 8480 71 00 00 (Прес-форми)
- **Статус:** НЕ ІСНУЄ в довіднику Medok
- **Правильний код:** `8480 71 00 90` - форми для лиття під тиском або компресійного формування: для гуми або пластмас: для лиття під тиском або компресійного формування: **інші**
- **Де зустрічається:** batch_003.csv та інші файли з прес-формами

### 1.2 Код 8207 30 00 00 (Штампи)
- **Статус:** НЕ ІСНУЄ в довіднику Medok
- **Правильний код:** `8207 30 10 90` - інструменти для пресування, штампування або вибивання: для оброблення металів: **інші**
- **Де зустрічається:** batch_003.csv та інші файли зі штампами

### 1.3 Код 9403 20 00 00 (Металеві меблі)
- **Статус:** НЕ ІСНУЄ - це код групи, а не повний 10-значний код
- **Правильні коди:**
  - `9403 20 20 00` - ліжка
  - `9403 20 80 00` - інші металеві меблі (стелажі, шафи, стійки)
- **Де зустрічається:** файли з металевими стелажами та шафами

## 2. ПОМИЛКИ КЛАСИФІКАЦІЇ

### 2.1 Прес-форми та штампи класифіковані як частини двигунів
- **Поточний код:** `8487 90 57 00` (частини машин без електричних з'єднувачів)
- **Проблема:** Код 8487 90 57 00 призначений для **частин двигунів внутрішнього згоряння**, а не для прес-форм чи штампів
- **Правильна класифікація:**
  - Прес-форми → `8480 71 00 90`
  - Штампи → `8207 30 10 90`

## 3. КОРЕКТНІ КЛАСИФІКАЦІЇ (підтверджено)

### 3.1 Ящики електроуправління
- **Код:** `8537 10 98 90`
- **Статус:** ІСНУЄ в довіднику
- **Опис:** Пульти, панелі, консолі, столи, розподільні щити та інші основи, обладнані двома або більше пристроями для контролю або розподілу електричного струму, для напруги не більш як 1000 В, інші

### 3.2 Автонавантажувачі (складські)
- **Код:** `8427 20 19 00`
- **Статус:** КОРЕКТНИЙ
- **Примітка:** Код `8427 20 11 00` призначений ЛИШЕ для навантажувачів **підвищеної прохідності** (rough terrain). Звичайні складські навантажувачі мають код `8427 20 19 00`

### 3.3 IP-телефони та телефонні апарати
- **Код для бездротових:** `8517 18 00 00` - телефонні апарати для бездротових мереж
- **Код для дротових:** `8517 11 00 00` - телефонні апарати для дротових мереж
- **Статус:** Обидва коди ІСНУЮТЬ

### 3.4 Джерела безперебійного живлення (ДБЖ/UPS)
- **Код:** `8504 40 30 00`
- **Статус:** ІСНУЄ
- **Опис:** Статичні перетворювачі: джерела безперебійного живлення

### 3.5 Статичні перетворювачі >7.5 кВА
- **Код:** `8504 40 88 00`
- **Статус:** ІСНУЄ
- **Опис:** Інші статичні перетворювачі потужністю понад 7,5 кВА

## 4. РЕКОМЕНДАЦІЇ

1. **Замінити неіснуючі коди** на валідні 10-значні коди з довідника Medok
2. **Переглянути класифікацію** прес-форм та штампів - вони НЕ повинні мати код 8487 90 57 00
3. **Перевірити всі записи** з кодом групи (менше 10 цифр) та доповнити до повного коду
4. **Для навантажувачів** - уточнити, чи є вони підвищеної прохідності (8427 20 11 00) чи звичайні складські (8427 20 19 00)
